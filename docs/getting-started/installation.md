# Installation

## Requirements

- Python 3.8+
- PyTorch 2.0+

## Install from PyPI

```bash
pip install faeyon
```

## Install from Source

```bash
git clone https://github.com/yourusername/faeyon.git
cd faeyon
pip install -e .
```

## Verify Installation

```python
import faeyon
print(faeyon.__version__)
```
